filabel Tool for labeling PRs at GitHub by globs.

Introduction

GitHub webhook service that adds labels to pull requests (PRs) by matching changes files with preconfigured patterns.

Usage

  1. Go to repository settings at GitHub
  2. Pick webhooks from left menu
  3. Press Add webhook
  4. Create webhook with configuration below and press Add webhook
